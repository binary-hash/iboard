👋 Touch Gestures Guide

  • 2-Finger Tap: Swap Pen & Eraser
  • 2-Finger Swipe (L/R): Cycle Backgrounds
  • 2-Finger Pinch: Adjust Brush Thickness
  • 3-Finger Tap: Clear Canvas
  • 3-Finger Swipe (Left): Undo
  • 3-Finger Swipe Down: Save Image
  • 3-Finger Swipe Up: Lock / Unlock Canvas
  • 4-Finger Tap: Toggle Zen Mode (Hide Toolbar)

Toggle gestures ON/OFF anytime with the ✋ button!
